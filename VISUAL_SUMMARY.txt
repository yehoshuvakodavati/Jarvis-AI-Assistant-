🔧 ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

WAKE WORD MODE:
  listen_for_wake_word()
  ├─ check_wake_word("computer")
  └─ return True/False

COMMAND MODE:
  listen_for_command(retry_count=2)
  ├─ listen(5s, 15s)
  ├─ validate_command()
  ├─ Retry on invalid
  └─ Return validated command

PROCESSING:
  handle_query(command)
  ├─ Decide search vs direct response
  ├─ execute (search, summarize, save)
  └─ Return response
